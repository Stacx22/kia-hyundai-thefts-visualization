# Kia & Hyundai Theft Trends Visualization

## Overview
Data storytelling project showing the rise in Kia and Hyundai vehicle thefts. Designed to help law enforcement, policymakers, and the public understand the scale and patterns of the issue.

## Data
- Public crime/theft datasets (e.g., city open data portals)
- Filtered to Kia and Hyundai theft incidents

## Visuals
- Pie and donut charts for make/model distributions
- Stacked bar charts for time or location splits
- Treemap for breakdowns by city/region
- Area/stacked area charts to show trends over time

## Repository Structure
- `code/` – R Markdown or scripts that generate the visuals
- `data/` – description of source datasets
- `docs/` – rendered R Markdown / PDF report
- `results/` – exported images of all charts

## How to Use
Open the report in `docs/` to see the narrative and visuals together.
